# Circuit Analyzer

一个网页端可视化电路仿真与分析工具。支持在浏览器中直接搭建并分析直流/交流稳态电路。

**在线地址：** [https://SemiCyrene.github.io/Circuit-Analyzer/](https://SemiCyrene.github.io/Circuit-Analyzer/)

---

## ⚡ 核心功能

*   **电路搭建**：支持电阻 (R)、电容 (C)、电感 (L)、直流源、交流源、四种受控源（VCVS/VCCS/CCCS/CCVS）及理想运放。
*   **连线模式**：支持“直角折线”与“直线/斜线”双模式，导线会记录其绘制模式并在元件拖动时自动拉伸。
*   **仿真求解**：采用改进节点电压法 (MNA)，支持直流 (DC) 和交流 (AC) 稳态分析。
*   **结果可视化**：引脚旁实时显示电压值（含幅值与相角），元件下方显示支路电流；AC 模式下支持在侧边栏绘制相量图。

## 🎹 键盘快捷键与交互

| 快捷键 / 操作 | 功能 |
| :--- | :--- |
| `S` | 选择工具 (Select) |
| `W` | 连线工具 (Wire) |
| `G` | 快速放置接地 (GND) |
| `D` | 删除工具 (Delete) |
| `R` | 旋转选中元件 90° |
| `Delete / Backspace` | 删除选中元件或导线 |
| `Escape` | 返回选择工具 / 取消当前动作 |
| `鼠标滚轮` | 画布缩放 (以鼠标为中心) |
| `右键按住拖拽` | 平移画布 |

## 📦 本地运行与构建

运行项目前请确保已安装 Node.js 环境：

```bash
# 1. 安装依赖
npm install

# 2. 启动本地开发服务
npm run dev

# 3. 生产打包
npm run build
```
